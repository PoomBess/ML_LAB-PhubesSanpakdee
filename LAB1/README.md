## เนื้อหาที่ทำ

- **LAB1: Dataset Exploration** — สำรวจข้อมูล ดูขนาด ชนิดข้อมูล สถิติสรุป ค่าที่หายไป ข้อมูลซ้ำ และการกระจายของ class
- **LAB2: Data Visualization** — วาด Histogram และ Correlation Heatmap
- **Part 3: Data Cleaning** — จัดการค่าที่หายไป ลบข้อมูลซ้ำ แก้ไขข้อมูลผิดปกติ แปลงชนิดข้อมูล
- **Part 4: Feature Engineering** — Label Encoding และ One-Hot Encoding
